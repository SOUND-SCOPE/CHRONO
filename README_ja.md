# CHRONO
### 舞台・音響・照明のキッカケ／Cueタイムライン可視化アプリ

**舞台・音響・照明スタッフのために作られたストップウォッチ＆Cueタイムライン可視化アプリです。**

> ⚠️ 本ソフトウェアは現在β開発版です。予期しない動作やエラーが発生する可能性があります。本番運用前に必ず動作検証を行ってください。

---

## 概要

CHRONOはリハーサル中にキッカケのタイミングを計測・記録・可視化し、本番では次のキッカケが近づくとカウントダウンと画面フラッシュでお知らせします。

> *「この音楽が始まって何秒後に照明を変える」*
> *「何秒後にナレーションが入る」*

そんなタイミングをリハーサルで計測し、タイムラインとして記録。あとはCHRONOに任せてください。

## 主な機能

- **TIMERモード** — リハーサル中にリアルタイムでラップタイムを計測。キッカケのタイミングでLAPボタンを押すだけ。
- **HISTORYモード** — 過去の計測セッションを一覧管理。グループ化・GAIN/FADER値の設定・リッチテキストメモの追加・キューシートとして印刷が可能。JSONによるデータ共有・エクスポートに対応。
- **PREVIEWモード** — 記録したタイムラインを再生。5秒前からカウントダウン（3・2・1・GO）と画面フラッシュで次のキッカケをお知らせ。
- **EDITモード** — ラップタイムやラベルをその場で直接編集。
- **REALTIMEモード** *(Enhanced / LTC)* — 事前に作成したセッションを再生しながらリアルタイムで上書き更新。
- **SETLIST** *(Enhanced / LTC)* — 複数のセッションをセットリストとして管理し、順番に切り替え。
- **LTC Sync** *(LTC限定)* — LTC（Linear Timecode）信号を受信してQ-LABなどのタイムコード環境と自動同期。

## CHRONOの特徴

✅ **サーバー不要** — 単一HTMLファイル。ブラウザがあればWindows・Mac・iPadどこでも動作。インストール不要。

✅ **画面フラッシュ** — キッカケの直前に画面全体がフラッシュ。手元の作業中でも見逃しません。

✅ **ビープ音・カスタム音源対応** — カウントダウン（PRE・3・2・1・GO）それぞれに個別の音声ファイルをアサイン可能。

✅ **データの持ち運び** — JSONファイルでエクスポート・インポート。複数デバイス間でデータを共有。

✅ **1カラム・2カラム表示切替** — 現場の状況に合わせてレイアウトを変更できます。
